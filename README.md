## 📋 المتطلبات

- **Node.js** نسخة 18 أو أحدث → [nodejs.org](https://nodejs.org)
- **حساب تطبيق Discord** → [discord.com/developers](https://discord.com/developers/applications)

---

## ⚙️ خطوات الإعداد

### 1️⃣ إنشاء البوت في Discord

1. افتح [Discord Developer Portal](https://discord.com/developers/applications)
2. اضغط **New Application** وسمّها
3. اذهب إلى **Bot** ← **Reset Token** واحفظ التوكن
4. فعّل هذه الصلاحيات تحت **Privileged Gateway Intents**:
   - ✅ `SERVER MEMBERS INTENT`
   - ✅ `MESSAGE CONTENT INTENT`

### 2️⃣ دعوة البوت للسيرفر

اذهب إلى **OAuth2 → URL Generator** واختر:
- Scopes: `bot`, `applications.commands`
- Permissions: `Administrator`

ثم افتح الرابط وأضف البوت لسيرفرك.

### 3️⃣ إعداد ملف config.js

افتح `config.js` وعدّل هذه القيم:

```js
token:    'توكن البوت',       // من Developer Portal
clientId: 'ID التطبيق',      // Application ID
guildId:  'ID السيرفر',      // كليك يمين على السيرفر ← Copy Server ID
```

#### إيجاد الـ IDs
> فعّل **Developer Mode** من: Settings → Advanced → Developer Mode
> ثم كليك يمين على أي قناة/رول/سيرفر واختر Copy ID

| الإعداد | كيف تجده |
|---------|----------|
| `logChannelId` | كليك يمين على قناة اللوق |
| `adminRoleId` | Server Settings → Roles → كليك يمين على الرول |
| `supportRoleId` | نفس أعلاه |
| `categoryId` | كليك يمين على الكاتيقوري |

### 4️⃣ تشغيل البوت

```bash
# تثبيت المكتبات
npm install

# تشغيل البوت
npm start
```

## 🤖 الأوامر

| الأمر | الصلاحية | الوظيفة |
|-------|----------|---------|
| `/setup` | أدمن | إرسال لوحة التذاكر في القناة الحالية |
| `/edit-panel` | أدمن | تعديل عنوان/وصف/صورة/لون الإمبد |
| `/close` | الكل | إغلاق التذكرة الحالية (مع تأكيد) |
| `/add @عضو` | أدمن | إضافة عضو للتذكرة |
| `/remove @عضو` | أدمن | إزالة عضو من التذكرة |
| `/rename اسم` | أدمن | تغيير اسم قناة التذكرة |
| `/ticket-info` | أدمن | عرض معلومات التذكرة |

## 🎛️ تعديل الإمبد بسهولة

### طريقة 1 — من config.js
عدّل قسم `panel` في ملف `config.js` ثم أعد تشغيل البوت وأرسل `/setup` من جديد.

### طريقة 2 — بدون إعادة تشغيل
استخدم الأمر `/edit-panel` مع ID رسالة الإمبد:
```
/edit-panel message-id: 1234567890  title: عنوان جديد  color: #ff0000
```

## 🔄 تدفق نظام التذاكر

```
عضو يضغط [🛒 شراء] أو [❓ استفسار]
    ↓
البوت يتحقق: هل عنده تذكرة مفتوحة؟
    ↓
ينشئ قناة خاصة ويضيف العضو + الأدمن
    ↓
يرسل إمبد ترحيب + زر [🔒 إغلاق]
    ↓
يرسل إشعار في قناة اللوق
    ↓
عند الإغلاق: يسأل هل أنت متأكد؟
    ↓
[✅ نعم] → يخفي القناة عن العضو
    ↓
يظهر للأدمن: [🔓 إعادة فتح] + [🗑️ حذف]
    ↓
عند الإعادة: يرسل للعضو "تم إعادة فتح تذكرتك"
```

---

## ⚠️ ملاحظات مهمة

- العضو **لا يستطيع** فتح أكثر من تذكرة واحدة في نفس الوقت
- بعد الإغلاق، **فقط الأدمن** يرى القناة ويتحكم فيها
- بعد الحذف، **لا يمكن الاسترداد**
- التذاكر تُحفظ في `tickets.json` وتُستعاد عند إعادة تشغيل البوت
